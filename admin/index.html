General Settings

This section will eventually allow configuration of basic portal settings such as title, URL, welcome message and login requirements.

Branding & Themes

This section will host the theme editor (colors, logos, light/dark mode).

Navigation Manager

This section will manage which pages appear in the portal navigation, their order and role‑based visibility.

Pages & Layouts

This section will contain the page builder UI, where admins can assemble pages from widgets.

Widget Library

This section will list core and custom widgets and their configuration.

Roles & Visibility

This section will control which pages and widgets are visible for which roles (end user, manager, local IT, etc.).

Developer Tools

This section will describe and eventually configure custom CSS/JS, integrations and widget SDK options.
